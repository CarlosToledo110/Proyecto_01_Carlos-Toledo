---Sistema de Evaluación de Contenidos de Streaming---
Este Proyecto consiste en un programa desarrollado 100% en C# que Simula el proceso de evaluación
De contenidos para una plataforma de streaming

Este Sistema permite analizar diferentes tipos de contenido (Películas, Series, Documentales o Eventos en vivo)
Tomando en cuenta factores como:
-Duración
-Horario de transmisión
-Nivel de producción
-Clasificación
Tomando como base estos datos, el programa determina si el contenido debe:
-Publicarse
-Publicarse con ajustes
-Enviarse a revisión
-Rechazarse
Además el sistema registra distintas estadísticas de uso mientras transcurre la sesión.
-Instrucciones para ejecutarlo
1. Abrir el proyecto en Visual Studio.
2. Compilar el programa.
3. Ejecutar el proyecto (tecla F5 o botón "Iniciar").
4. Aparecerá un menú en consola.
5. Ingresar el número de la opción deseada y presionar Enter.

Opciones disponibles:
-Evaluar contenido
-Mostrar reglas
-Mostrar estadísticas
-Reiniciar estadísticas
-Salir

El programa fue desarrollado utilizando estructuras básicas de programación en C#, incluyendo:
-switch para el control del menú
-do-while para mantener el sistema en ejecución
-if y if-else para la toma de decisiones
-while para validar entradas del usuario
-for para cumplir con iteraciones dentro del sistema

El sistema sigue una lógica de evaluación basada en reglas previamente definidas,
Las cuales permiten clasificar el impacto del contenido y tomar una decisión final.
